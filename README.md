# Inventario_de_Laboratorio_tech
Sistema de Inventário em Java focado em POO Avançada e I/O. O projeto aplica herança e polimorfismo para gerenciar equipamentos e computadores, realizando a leitura, validação e persistência de dados em arquivos TXT através de BufferedWriter e BufferedReader, com tratamento robusto de exceções e geração automatizada de logs.
